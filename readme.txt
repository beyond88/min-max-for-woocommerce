== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin zip file to your plugin directory ‘/wp-content/plugins/’
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Go to product details and enable minimum and maximum quantity
4. Save/update product details
5. Set custom message for minimum and maximum quantity
